# **App Name**: zennsmm

## Core Features:

- Live API Synchronizer: Automated data-fetching engine that connects to smmzio.com/api/v2 to keep services, descriptions, and INR prices perfectly in sync with the provider.
- Social Boost Planner Tool: A generative AI-powered tool that helps users select the best marketing packages based on their specific account growth goals and budget.
- Platform-Specific Catalog: Dynamic service grid with dedicated category filters and instant-search functionality optimized for finding SMM solutions quickly on mobile.
- Quick-Form Ordering System: Mobile-first order placement UI featuring link validation, dynamic quantity math, and real-time total calculation in Indian Rupees (₹).
- Merchant Readiness Suite: Pre-configured landing and legal pages (Refunds, TOS, Privacy) designed to meet Razorpay and bank verification requirements for Indian businesses.
- Responsive User Hub: Compact, login-based navigation allowing users to view order histories and account balances without overwhelming smaller smartphone screens.
- Secure Gateway Middleware: Secure server-side handling of external API keys to process orders and track delivery status safely.

## Style Guidelines:

- Primary Color: Professional Deep Blue (#5193fa) chosen to evoke trust and established digital presence.
- Background Color: Nocturnal Navy (#0c0e13) providing a sophisticated, low-glare foundation for the dark theme.
- Accent Color: Vibrant Electric Cyan (#51e6fa) for critical CTAs and active state highlights to ensure visual hierarchy on mobile devices.
- Headline Font: 'Space Grotesk' for a precise, tech-forward aesthetic; Body Font: 'Inter' for superior readability and objective information delivery.
- Compact, vertical-centric grid that prioritizes text-wrapping and oversized touch targets for seamless Android and iPhone operation.
- Fast-tempo entrance transitions and persistent loading skeletons to maintain a perception of high performance during data fetching.
- Simplified monochrome glyphs for various social media networks to maintain the minimalist and trusted agency look.
